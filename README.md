# Voice Command Canvas

Say a number out loud — the app draws that many apples on the canvas using voice recognition

🎨 **[Try it live](https://creativemind012.github.io/VOICE-COMMAND-CANVAS/)**

## How it works
Uses the browser's Web Speech API to capture voice input and p5.js to draw on the canvas based on recognised commands.

## Tech
- Web Speech API — voice recognition
- [p5.js](https://p5js.org/) — canvas drawing

## Run locally
Open `index.html` in Chrome (Web Speech API works best in Chrome). Allow microphone access when prompted.
